# テスト共通規約

## テストデータのタスク名

テストデータでタスク名を指定する場合は、寿司ネタを使用してください。

例: `マグロ`, `サーモン`, `エビ`, `イカ`, `タマゴ`, `ウニ`, `イクラ`, `アナゴ`, `ホタテ`, `カンパチ`

## テスト実行ログ

各テストスイート（`describe` ブロック）の開始時と終了時にログを出力してください。

```typescript
beforeAll(() => {
  console.log('[START] <テストスイート名>');
});

afterAll(() => {
  console.log('[END] <テストスイート名>');
});
```
