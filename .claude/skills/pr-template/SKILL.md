# PR テンプレート

## it の説明文は関西弁

テストコードの `it` の説明文は関西弁で記述してください。

例:
```typescript
it('タスクが正しく作成されるんやで', async () => { ... });
it('存在しないIDやったら404返すねん', async () => { ... });
it('バリデーションエラーで400になるっちゅうねん', async () => { ... });
```

## PR タイトル

PRタイトルは「〇〇の冒険」という形式にしてください。

例: `rules/skills使用ver. tasksテストの冒険`
